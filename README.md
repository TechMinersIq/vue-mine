
# VUE MINE [![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)

VUE MINE is a simple vue project template.


## Tech Stack

* node v18.13.0
* vue v3.2.47
* vuetify v3.0.0-beta.0
* pinia v2.0.32
* axios v1.3.4
* vue-router v4.1.6
* vue-i18n v9.2.2


## Installation

```bash
  npm install
```

## Start

```bash
  npm run dev
```
    
## Build

```bash
  npm run build
```
