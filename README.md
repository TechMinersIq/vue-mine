
# VUE MINE [![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)

VUE MINE is a simple vue project template.


## Tech Stack

* node v18.16.1
* vue v3.3.4
* vuetify v3.3.12
* pinia v2.1.6
* axios v1.4.0
* vue-router v4.2.4
* vue-i18n v9.2.2


## Installation

```bash
  npm install
```

## Start

```bash
  npm run dev
```
    
## Build

```bash
  npm run build
```

## Electron

```bash
  npm run electron:start
```

## Package Electron

```bash
  npm run package-win
```

```bash
  npm run package-linux
```

```bash
  npm run package-mac
```
